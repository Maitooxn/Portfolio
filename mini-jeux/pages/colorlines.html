Color Lines

Un jeu de réflexion où vous devez aligner des billes de même couleur

À propos du jeu

Color Lines est un jeu de réflexion où le joueur doit aligner des billes de même couleur pour les faire disparaître et marquer des points. À chaque tour, de nouvelles billes apparaissent aléatoirement sur le plateau. Le jeu se termine lorsque le plateau est rempli.

Comment jouer

  1. Téléchargez le fichier Python du jeu en cliquant sur le bouton ci-dessous.
  2. Installez Python si ce n'est pas déjà fait (version 3.6 ou supérieure recommandée).
  3. Installez Tkinter si nécessaire (généralement inclus avec Python).
  4. Exécutez le fichier en double-cliquant dessus ou via la commande python colorlines.py dans un terminal.
  5. Jouez : Cliquez sur une bille pour la sélectionner, puis cliquez sur une case vide pour la déplacer. Formez des lignes de 5 billes ou plus de même couleur pour les faire disparaître et marquer des points.

Configuration requise

  • Python 3.6 ou supérieur
  • Tkinter (bibliothèque graphique, généralement incluse avec Python)
  • Système d'exploitation : Windows, macOS ou Linux

Fonctionnalités

  • Plateau de jeu dynamique
  • Algorithme de recherche de chemin pour déplacer les billes
  • Détection automatique des alignements
  • Système de score
  • Interface graphique intuitive avec Tkinter

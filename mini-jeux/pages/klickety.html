Klickety

Un jeu de puzzle où vous éliminez des groupes de blocs de même couleur

À propos du jeu

Klickety est un jeu de puzzle où le joueur doit éliminer des groupes de blocs de même couleur. Lorsqu'un groupe est éliminé, les blocs au-dessus tombent pour remplir les espaces vides. L'objectif est de vider le plateau ou d'obtenir le score le plus élevé possible.

Comment jouer

  1. Téléchargez le fichier Python du jeu en cliquant sur le bouton ci-dessous.
  2. Installez Python si ce n'est pas déjà fait (version 3.6 ou supérieure recommandée).
  3. Installez Tkinter si nécessaire (généralement inclus avec Python).
  4. Exécutez le fichier en double-cliquant dessus ou via la commande python klickety.py dans un terminal.
  5. Règles du jeu :
    • Cliquez sur un groupe de blocs adjacents de même couleur pour les éliminer.
    • Plus le groupe est grand, plus vous marquez de points.
    • Après chaque élimination, les blocs au-dessus tombent pour remplir les espaces vides.
    • Le jeu se termine lorsqu'il n'y a plus de groupes possibles à éliminer ou lorsque le plateau est vide.
    • L'objectif est d'obtenir le score le plus élevé possible.

Configuration requise

  • Python 3.6 ou supérieur
  • Tkinter (bibliothèque graphique, généralement incluse avec Python)
  • Système d'exploitation : Windows, macOS ou Linux

Fonctionnalités

  • Grille de jeu colorée
  • Détection des groupes de blocs adjacents
  • Animation de chute des blocs
  • Système de score basé sur la taille des groupes éliminés
  • Plusieurs niveaux de difficulté

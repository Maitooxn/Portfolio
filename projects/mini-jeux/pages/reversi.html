Reversi

Un jeu de stratégie classique aussi connu sous le nom d'Othello

À propos du jeu

Reversi (ou Othello) est un jeu de plateau à deux joueurs où l'objectif est d'avoir plus de pions de sa couleur que l'adversaire à la fin de la partie. À chaque tour, un joueur place un pion qui capture les pions adverses situés entre ce nouveau pion et un autre pion de sa couleur.

Comment jouer

  1. Téléchargez le fichier Python du jeu en cliquant sur le bouton ci-dessous.
  2. Installez Python si ce n'est pas déjà fait (version 3.6 ou supérieure recommandée).
  3. Installez Tkinter si nécessaire (généralement inclus avec Python).
  4. Exécutez le fichier en double-cliquant dessus ou via la commande python reversi.py dans un terminal.
  5. Règles du jeu :
    • Le jeu se joue sur un plateau 8×8.
    • Les joueurs placent à tour de rôle un pion de leur couleur sur le plateau.
    • Un coup n'est valide que s'il permet de capturer au moins un pion adverse.
    • La capture se fait en encadrant une ligne de pions adverses entre deux pions de sa couleur.
    • Les pions capturés changent de couleur.
    • Le jeu se termine quand le plateau est plein ou qu'aucun joueur ne peut plus jouer.
    • Le joueur avec le plus de pions de sa couleur gagne.

Configuration requise

  • Python 3.6 ou supérieur
  • Tkinter (bibliothèque graphique, généralement incluse avec Python)
  • Système d'exploitation : Windows, macOS ou Linux

Fonctionnalités

  • Plateau de jeu 8x8
  • Mode deux joueurs
  • Indication des coups possibles
  • Comptage automatique des pions
  • Détection de fin de partie
